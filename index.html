Meus Projetos de Confeitaria

Este projeto é uma página web desenvolvida com HTML e CSS, dedicada a apresentar uma coleção dos meus bolos favoritos. A ideia é compartilhar receitas especiais e inspirações visuais para quem ama confeitaria e design de sobremesas.

O site organiza os bolos em uma galeria visual atrativa, com fotos destacadas, descrições das receitas e detalhes como ingredientes principais, tempo de preparo e dicas de finalização.

O CSS aplica uma paleta em tons pastéis, tipografia personalizada do Google Fonts e efeitos suaves em botões e links. O layout é responsivo, adaptando-se bem a celulares e tablets.
